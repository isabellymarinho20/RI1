# RI1 - Marvel 

###Descrição da RI1:
Este projeto utiliza JavaScript para criar um simulador de combate e comparação de atributos entre os **Vingadores** e seu maior rival, **Thanos**. O foco é aplicar lógica de programação para analisar  quem possui vantagem para vencer o Thanos.

**Principais implementações:**
* **Manipulação de Arrays:** Armazenamento centralizado de todos os heróis.
* **Criação de Objetos:** Uso de classes para padronizar os atributos dos personagens.
* **Estruturas de Repetição:** Uso do laço `for` para percorrer a lista de personagens e realizar as comparações.
* **Lógica de Comparação:** Verificação dinâmica de atributos numéricos para determinar em quais pontos um herói supera o vilão.

### Tecnologias:
* **Linguagem:** JavaScript.
* **Ambiente de Execução:** Node.js.
* **Versionamento:** Git.
